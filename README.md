# **DecodeLabs Summer Internship - Project 3**

For the third project of the DecodeLabs frontend internship, the focus shifted from CSS layouts to actual interactivity using JavaScript. To get a solid grip on how the Document Object Model (DOM) works, I built a simple, clean tally counter application, complete with a dynamic dark/light theme toggle.

> **[View Live Demo Here](https://rounak-56.github.io/decodelabs-project-3/)**

<br>

## Project Objective
Instead of just making things look good, the goal for this milestone was to make things *work*. The main objective was to understand the Input-Process-Output loop: listening for a user's click, using JavaScript to update the data, and dynamically changing the text and visual themes on the screen without refreshing the page.

<br>

## Tech Stack & Skills Highlighted
* **JavaScript (Vanilla):** Wrote the core logic to handle state management (incrementing, decrementing, resetting the counter, and toggling the current theme).
* **DOM Manipulation:** Safely updated HTML elements in real-time using `textContent` and manipulated CSS classes with `classList.toggle()` for the theme switch.
* **Event Listeners:** Wired up the buttons so they actually trigger the correct JavaScript functions when a user interacts with them.
* **Responsive Design:** Used CSS media queries to ensure the interface scales perfectly from desktop monitors down to mobile screens.
* **Theme Management:** Implemented a modern Light/Dark mode switch to handle dynamic visual state changes.
